# Kendy MCP Assistant

An AI-powered personal assistant with multi-tenant support, voice processing capabilities, and MCP (Model Context Protocol) integration.

## Overview

Kendy MCP Assistant is a comprehensive platform that combines:
- **MCP Protocol Client**: Integration with Model Context Protocol for AI-powered assistance
- **Voice Processing**: Speech-to-Text (STT) and Text-to-Speech (TTS) with Vietnamese language support
- **Multi-Tenant Architecture**: Isolated account systems with per-tenant MCP server management
- **Mobile-First**: React Native mobile application for iOS and Android
- **Backend Services**: Scalable backend infrastructure with database support
- **AI Planning**: Intelligent planning and reminder system
